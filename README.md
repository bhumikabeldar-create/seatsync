# Seatsync Project

## Overview
Seatsync is a project aimed at streamlining the process of managing seating arrangements and reservations for events. It integrates various functionalities for users to easily select and book seats based on their preferences.

## Features
- User-friendly interface for booking seats
- Real-time availability updates
- Multi-event support
- Admin panel for managing events and reservations

## Setup Instructions

### Prerequisites
- Node.js (v14 or later)
- npm (v6 or later) or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/bhumikabeldar-create/seatsync.git
   ```

2. Navigate into the project directory:
   ```bash
   cd seatsync
   ```

3. Install dependencies:
   ```bash
   npm install
   ```
   or
   ```bash
   yarn install
   ```

### Running the Application
To start the application, run:
```bash
npm start
```

### Testing
To run the tests, use:
```bash
npm test
```
